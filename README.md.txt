# Day 10 - Python File Handling (Write to File)

## 📌 Overview
This project demonstrates how to write data into a text file using Python File Handling.

## 📚 Topics Covered
- open()
- with open()
- Append Mode ("a")
- write()
- File Handling
- User Input
- f-strings

## 🚀 Project
Student Record Writer

The program accepts student details from the user and stores them in a text file named `students.txt`.

## Sample Output

Enter Student ID: 101
Enter Name: Akhil
Enter Age: 20
Enter Course: CSE
Enter Marks: 90

Student Record Saved Successfully!

## Sample File

101,Akhil,20,CSE,90
102,Rahul,21,ECE,88
